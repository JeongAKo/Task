## SK Planet API이용 날씨앱 만들기



Open API를 이용하여 서버와 통신하고 JSON Data를 받아 필요한 정보만 파싱하여 데이터를 다뤘습니다.

JSONSerialization과 failable initializer를 사용하여 fatch service 데이터 파싱을 하였습니다.
